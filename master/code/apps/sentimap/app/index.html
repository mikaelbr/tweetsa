SentiMap: Sentiment on Twitter across the United States

A Twitter Sentiment Analysis tool developed for a Master's Thesis on NTNU. This tool uses New Signatures U.S. Map jQuery Plugin for the map, and Highcharts for charting. The sentiment data used is classified by a server developed for this Master's thesis.
